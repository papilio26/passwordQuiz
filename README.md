# パスワードセキュリティクイズ

弱いパスワードの危険性を学ぶための教育用Webアプリケーションです。

## 概要
10問のクイズを通じて、どのようなパスワードが危険か、なぜ危険なのかを学べます。

## 作成目的
TryHackMeやセキュリティ関連の学習を通じて、弱いパスワードがいかにサイバー攻撃の入り口になりやすいかを学びました。
この知識を実践的に伝えるため、クイズ形式のWebアプリを作成しました。

## デモ
https://papilio26.github.io/passwordQuiz/

## 使用技術
- HTML
- CSS
- JavaScript
- Web Audio API（音のフィードバック）
